PROJECT_CASE_STUDY

01_ Projet prototype

Atelier Berger
Carte Interactive

Prototype web interactif permettant d’explorer les projets de l’Atelier Berger à travers un globe 3D, une liste de réalisations et des fiches détaillées.

Voir le projet → GitHub ↗

Role

UI design, prototype interactif, intégration React, globe 3D, déploiement

Stack

React, TypeScript, Vite, Tailwind CSS, Motion, React Globe GL, Three.js/WebGL, Google AI Studio / Stitch, Vercel

Objectif

Présenter les projets architecturaux de l’Atelier Berger dans une expérience plus immersive, visuelle et interactive qu’un portfolio classique.

Statut

Prototype fonctionnel déployé en ligne.

Le problème

Les projets d’architecture sont souvent présentés sous forme de grilles statiques. Cela rend difficile la perception de leur répartition géographique, de leur diversité et de leur contexte.

La solution

Le projet propose une interface interactive avec un globe 3D, une liste de projets et des fiches détaillées. L’utilisateur peut explorer les réalisations par localisation, visualiser les projets sur la carte et accéder rapidement aux informations principales.

Workflow

1. Prototype avec Google AI Studio / Stitch

Le projet a d’abord été généré et structuré avec Google AI Studio / Stitch afin d’obtenir rapidement une base d’interface, des composants React et une première logique de navigation.

2. Structuration du projet

L’application a ensuite été organisée en composants : WorldMap, ProjectList, ProjectDetail, ContactModal et CreativeProcessStory.

3. Intégration du globe 3D

Le globe a été développé avec React Globe GL, une librairie React basée sur Three.js et WebGL. Les projets sont affichés sous forme de points géolocalisés grâce à leurs coordonnées latitude/longitude.

4. Design et interactions

L’interface utilise Tailwind CSS pour la mise en page et Motion pour les animations. L’objectif était de créer une expérience fluide, visuelle et premium adaptée à un studio d’architecture.

5. Déploiement

Le projet a été versionné sur GitHub puis déployé avec Vercel pour obtenir une version consultable en ligne.

Fonctionnalités

Globe interactif

Visualisation des projets sur un globe 3D avec points cliquables.

Liste de projets

Affichage des réalisations avec leurs informations principales.

Fiches détaillées

Chaque projet peut être consulté à travers une fiche dédiée.

Navigation immersive

L’utilisateur peut passer d’une vue globale à une lecture plus détaillée des projets.

Ce que ça montre

Ce projet montre ma capacité à transformer un prototype généré avec IA en interface web fonctionnelle : structuration React, intégration TypeScript, design responsive, visualisation 3D, logique de composants, gestion des données projets et déploiement web.
